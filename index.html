Configuración de Google Sheets

Sigue estos pasos una sola vez para conectar la app con tu hoja de cálculo.

Pasos de configuración
1

Crea una hoja de cálculo en Google Sheets

Abre Google Sheets y crea una hoja nueva. Ponle el nombre que quieras, por ejemplo Elios3_Reportes. Asegúrate de estar conectado con tu cuenta de Google.

2

Abre el editor de Apps Script

Dentro de la hoja, ve al menú Extensiones → Apps Script. Se abrirá el editor de código de Google.

3

Pega el siguiente código

Borra todo el contenido del editor y pega exactamente este código. Luego guarda con Ctrl+S:

const SHEET_NAME = "Reportes"; function doPost(e) { try { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); sheet.appendRow([ "N° Reporte","Fecha Inspección","Fecha Generación", "Empresa","RUT Empresa","Industria","Ubicación", "Activo","Tag Activo","Tipo Activo","Material", "Descripción","Objetivo","Entregables","Tipo Inspección","Confidencial", "Piloto","Email Piloto","Empresa Operadora", "Inspector","Contacto Sitio","Teléfono Sitio", "N° Vuelos","Riesgo Inicial","Riesgo Residual","Nivel Riesgo", "Factores Alto","Factores Medio","Factores Bajo" ]); } const d = JSON.parse(e.postData.contents); sheet.appendRow([ d.reportNum, d.inspDate, d.generatedAt, d.company, d.companyId, d.industry, d.location, d.assetName, d.assetTag, d.assetType, d.assetMaterial, d.assetDesc, d.inspGoal, d.deliverables, d.inspMode, d.confidential, d.pilotName, d.pilotEmail, d.operatorCompany, d.inspectorName, d.siteContact, d.sitePhone, d.numFlights, d.initialScore, d.residualScore, d.riskLevel, d.highFactors, d.medFactors, d.lowFactors ]); return ContentService.createTextOutput( JSON.stringify({ok:true}) ).setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput( JSON.stringify({ok:false, error:err.message}) ).setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { try { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) return ContentService.createTextOutput( JSON.stringify({ok:true, rows:[]}) ).setMimeType(ContentService.MimeType.JSON); const data = sheet.getDataRange().getValues(); const headers = data[0]; const rows = data.slice(1).map(r => { const obj = {}; headers.forEach((h,i) => obj[h] = r[i]); return obj; }); return ContentService.createTextOutput( JSON.stringify({ok:true, rows}) ).setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput( JSON.stringify({ok:false, error:err.message}) ).setMimeType(ContentService.MimeType.JSON); } }
4

Despliega como aplicación web

En el editor de Apps Script, haz clic en Implementar → Nueva implementación. Selecciona tipo Aplicación web. En el campo "Quién tiene acceso" elige Cualquier usuario. Haz clic en Implementar y autoriza los permisos cuando te lo solicite.

5

Copia la URL de la aplicación web y pégala aquí

Después de implementar, Google te mostrará una URL que empieza con https://script.google.com/macros/s/.... Cópiala y pégala abajo:

✅ URL guardada correctamente.
⚠ La URL no parece válida. Verifica que empiece con https://script.google.com
6

¡Listo! Prueba guardando tu primera evaluación

Vuelve a la pestaña Nueva evaluación, completa el formulario y haz clic en "Guardar en Google Sheets". El reporte aparecerá automáticamente en tu hoja de cálculo.
